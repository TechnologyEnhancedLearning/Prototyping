Register multiple delegates

To register multiple people, download the template and add each person as a new row.

You'll be able to enrol all of them onto self assessments or promote them into admin roles as part of this process.

If you want to enrol or promote groups of delegates differently, you should complete a separate template for each group.

If you're unsure how to complete any of the columns, you can refer to the reference sheets on the additional tabs within the template.

Download template

Once you have filled in the template, save it locally on your computer.

Select Choose file and select your completed template
